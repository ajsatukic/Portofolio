Graduation Thesis

The topic "Basics of Robotics and Lego Mindstorms Software" explores fundamental concepts of robotics, its historical development, and modern applications. The work reviews the essential components of robots, their classifications, and key areas of application, such as industry and medicine. It emphasizes Lego Mindstorms sets as an educational tool for robot development, including the programmable EV3 module. The paper describes the use of Lego Mindstorms software for robot programming, highlighting features like sensors, motors, and coding blocks through an intuitive graphical interface.

Millionaire

This application is a quiz inspired by the "Who Wants to Be a Millionaire?" format, where users answer multiple-choice questions to win monetary rewards displayed on a value pyramid. The start screen allows users to enter their name, and each question is timed with an animated countdown. The app displays questions and answers with visual feedback for correct and incorrect options, while the pyramid highlights the current reward level. At the end of the game, the user sees the total amount earned. Built with React, the app features a modern design and includes components such as a timer, quiz questions, and a reward pyramid for an engaging experience.

Advanced Data Structures for Efficient Element Manipulation

The project focuses on the implementation of two advanced generic data structures. The first structure enables quick access, insertion, and removal of the median element in a collection, using two heaps (max-heap and min-heap) to maintain a balance between smaller and larger elements. The second structure, a double heap, allows efficient removal of the smallest and largest elements while adhering to specific organizational rules based on the depth of elements in the heap tree. Both structures are implemented from scratch using arrays and are adaptable through custom comparators, making them flexible for various data types. The project emphasizes the practical understanding of data structures and algorithms.
